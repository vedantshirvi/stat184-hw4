# Project and Repo Plan

## Project Plan

### Goal
Complete a reproducible data analysis report for STAT 184 Activity 15.

### Needs
- R and RStudio installed
- Quarto installed
- Required R packages loaded

### Steps
1. Load and explore the dataset
2. Create visualizations and tables
3. Write narrative text describing findings
4. Render the QMD file to PDF

## Repo Maintenance Plan

### Goal
Keep the repo organized and up-to-date using good version control practices.

### Needs
- A main branch that stays clean and up-to-date
- A dev branch for active work
- Issues to track tasks

### Steps
1. Do all work on the dev branch
2. Use issues to track tasks
3. Write meaningful commit messages
4. Merge to main via pull requests when work is complete
